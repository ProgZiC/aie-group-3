## 5. Analysis

### 5.1 Сравнение алгоритмов (важные наблюдения)

- **Где KMeans "ломается" и почему?**  
  KMeans показал умеренное качество на Dataset B (Silhouette=0.307) из-за нелинейной структуры данных, что противоречит его предположению о сферичности кластеров.

- **Где DBSCAN/иерархическая кластеризация выигрывают и почему?**  
  В данном эксперименте DBSCAN не показал преимуществ:
  - На Dataset A KMeans был лучше.
  - На Dataset B DBSCAN нашел много шума (7.24%) и низкое качество.
  - На Dataset C DBSCAN не справился с разной плотностью, помечая 91.23% точек как шум.

- **Что сильнее всего влияло на результат?**  
  1. **Масштабирование признаков** – критично для Dataset A с разными шкалами.
  2. **Разная плотность кластеров** – проблема для DBSCAN на Dataset C.
  3. **Нелинейная структура** – сложность для обоих алгоритмов на Dataset B.
